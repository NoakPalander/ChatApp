Q_SIGNAL void AppWidget::Append ( QString const &  text)

Invoked internally when text wants to be appended to the chat box.

Attention
This is not a normal function, it has no implementation, it's a Qt signal
Parameters
textthe text that should be written to the chatbox

It's mainly invoked from the server/client callbacks, however these are run on a separate thread managed by the processor's io_service, thus we can't perform draws within the callbacks, hence we emit a signal so Qt can perform draws on the UI thread.
